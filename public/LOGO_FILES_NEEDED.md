# Logo Files Needed

Place the following logo files in this `/public/` folder:

## Required Files

### 1. Main Logo
- **`logo.svg`** (preferred) or **`logo.png`**
  - Size: 200-400px wide
  - Used in: Login page, Dashboard header
  - Format: SVG recommended (scalable, no blur)

### 2. Favicon (Browser Tab Icon)
- **`favicon.ico`**
  - Size: 32x32px
  - Used in: Browser tab icon
  - Format: ICO format

### 3. PWA Icons (Progressive Web App)
- **`icon-192.png`**
  - Size: 192x192px
  - Format: PNG with transparent or colored background

- **`icon-512.png`**
  - Size: 512x512px
  - Format: PNG with transparent or colored background

- **`apple-touch-icon.png`**
  - Size: 180x180px
  - Format: PNG
  - Used when app is added to iOS home screen

## After Adding Logo Files

1. Place all files in `/public/` folder
2. Restart dev server: `pnpm run dev`
3. Hard refresh browser: `Ctrl+Shift+R` (Windows/Linux) or `Cmd+Shift+R` (Mac)
4. Check:
   - Browser tab icon (favicon)
   - Login page logo
   - Dashboard header logo
